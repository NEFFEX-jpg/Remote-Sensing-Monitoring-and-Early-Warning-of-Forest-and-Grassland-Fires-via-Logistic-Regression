# 基于逻辑回归的林草火灾遥感监测预警系统

## 📋 项目简介

本项目利用**逻辑回归（Logistic Regression）**算法，结合遥感数据和气象数据，构建森林和草原火灾风险预测模型，实现对林草火灾的**早期预警**和**风险等级评估**。

系统通过分析多种影响因子（植被含水量、地形、气象条件等），分别建立森林和草原的火险预测模型，并生成可视化的**火险概率地图**，为火灾防控决策提供科学依据。

---

## 🎯 研究目标

1. 构建基于逻辑回归的林草火灾预测模型
2. 分析影响火灾发生的关键因子及其显著性
3. 实现对遥感影像的火险概率预测
4. 生成直观的火险等级分布图

## 📊 数据说明

### 训练数据
- **样本量**：99,670 条记录
- **标签**：`tag`（0=未着火，1=着火）
- **土地类型**：`ZGBP`（1-8 为森林，9-11 为草原）

### 特征变量

| 特征 | 说明 | 类型 |
|------|------|------|
| LFMC16 | 16天合成植被含水量 | 连续 |
| LFMC8 | 8天合成植被含水量 | 连续 |
| 16_8dif | LFMC16与LFMC8的差值 | 连续 |
| LAI | 叶面积指数 | 连续 |
| altitude | 海拔高度 | 连续 |
| slope | 坡度 | 连续 |
| aspect | 坡向 | 离散（独热编码） |
| wind | 风速 | 连续 |
| temperature | 温度 | 连续 |
| rain | 降水量 | 连续 |
| humid | 空气湿度 | 连续 |

### 测试数据
- **格式**：GeoTIFF（.tif）
- **来源**：MODIS 遥感影像
- **内容**：11 个图层对应上述特征

## 🔬 技术路线

```
数据加载 → 数据预处理 → 特征工程 → 模型训练 → 模型评估 → 遥感预测 → 结果可视化
```

### 详细流程

1. **数据预处理**
   - 缺失值处理（删除含缺失值的样本）
   - 坡向离散化（划分为8个方位）
   - 独热编码处理分类变量

2. **模型训练**
   - 按土地类型分离森林/草原样本
   - 8:2 划分训练集/测试集
   - 特征标准化（StandardScaler）
   - 逻辑回归训练（L2正则化，类别平衡）

3. **模型评估**
   - 分类报告（Precision、Recall、F1-score）
   - ROC曲线与AUC值
   - 显著性差异分析（t检验）

4. **遥感预测**
   - 读取多波段TIF影像
   - 批量预测火险概率
   - 合并森林/草原预测结果
   - 生成火险概率地图
